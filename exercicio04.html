Seletores

Seletores da Tag

Ao utilizar o seletor tipo tag em H1 e H2 todos os elementos deste tipo vão herdar os estilos definidos pelo código CSS

Seletores tipo.id

O seletor de tipo #id deve ser utilizado uma única vez no arquivo! Cada elemento pode ter apenas um id. Cada página pode apenas um elemento com aquele id.

Seletor de classe

Qualquer informação de estilo que precise ser aplicada a múltiplos elementos em uma página deve ser feita com uma classe. Você usar a mesma classe para vários elementos. você pode usar várias classe para um mesmo elemento.
